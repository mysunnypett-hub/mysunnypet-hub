### 🐾 Sobre a My Sunny Pet

A **My Sunny Pet** é uma plataforma dedicada a oferecer serviços de pet grooming e cuidados especializados diretamente no domicílio do cliente. Nosso foco é unir conveniência para os tutores e um ambiente livre de estresse para os pets.

- 🏠 **Serviço:** Pet Shop Domiciliar Premium.
- 💻 **Tech Stack:** Automação de agendamentos e experiência do cliente via IA.
- 📍 **Atendimento:** Rio de Janeiro, RJ.
- 📧 **Contato:** [fernandoborgescerqueira@gmail.com](mailto:fernandoborgescerqueira@gmail.com).
